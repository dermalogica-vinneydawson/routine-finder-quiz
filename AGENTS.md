# Routine Finder Quiz - Agent Instructions

## Project Context

This repository contains the **Dermalogica Routine Finder Quiz** - an interactive quiz that guides users to their personalized skincare routine recommendations.

1. **Understand the domain**: Skincare routines, product recommendations, and user skin-type assessment
2. **Preserve brand voice**: Dermalogica is professional, educational, and skincare-focused
3. **Quiz flow**: Consider the user journey through questions and how answers map to recommendations
4. **Accessibility**: Ensure quiz interactions work for all users (keyboard, screen readers, etc.)
